Listas

Los documentos rtf, de texto enriquecido, tienen la posibilidad de dividir documentos en apartados o listas sin numerar. Aqui aparece algun ejemplo:

Tipo de lista numerada:

  1. Elemento de nivel cero, cero
    1. Elemento de nivel uno, cero
    2. Elemento de nivel uno, uno
      1. Elemento de nivel dos, cero
      2. Elemento de nivel dos, uno
  2. Elemento de nivel cero, uno
  3. Elemento de nivel cero, dos

Tipo de lista no numerada

  • Elemento de nivel cero, cero
    • Elemento de nivel uno, cero
      • Elemento de nivel dos, cero
      • Elemento de nivel dos, uno
    • Elemento de nivel uno, uno
  • Elemento de nivel cero, uno
    • Elemento de nivel uno dos
